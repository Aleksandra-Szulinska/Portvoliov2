The organizational space was built with the intention of being user-friendly on mobile phones.

It was designed based on my personal needs, which I believe will also be useful for many others. The layout features four options and a transparent design, and a personalisation option, which makes it easy to access everything in one place. This project is built using Javascript and has the ability to save data on HTML inner storage, ensuring all your information is stored safely. Additionally, the sheet is password-protected for your convenience.
